# 📦 Archive Folder (CrystalCastle)

## Purpose / วัตถุประสงค์
**English:**  
This folder contains files, workflows, or components that are outdated, replaced, or scheduled for deletion.  
Contributors should avoid using these files and instead refer to updated versions in `src/` or `docs/`.

**ไทย:**  
โฟลเดอร์นี้เก็บไฟล์, workflows, หรือ components ที่ล้าสมัย ถูกแทนที่แล้ว หรือควรถูกลบออก  
Contributor ไม่ควรใช้งานไฟล์เหล่านี้ และควรอ้างอิงเวอร์ชันใหม่ใน `src/` หรือ `docs/`

---

## Guidelines / แนวทาง
- **English:**  
  - Do not reference files in `archive/` for new PRs.  
  - If you find a file here still in use, migrate it to the updated structure.  
  - Reviewers should flag PRs that reintroduce archived files.

- **ไทย:**  
  - ห้ามอ้างอิงไฟล์ใน `archive/` สำหรับ PR ใหม่  
  - หากพบว่ายังมีการใช้งานไฟล์ในนี้ ให้ย้ายไปโครงสร้างใหม่  
  - Reviewer ควรแจ้งเตือน PR ที่นำไฟล์ archive กลับมาใช้อีก
