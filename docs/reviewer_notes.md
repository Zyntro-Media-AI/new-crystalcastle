# CrystalCastle Reviewer Action Checklist (TH/EN)

## 🔵 Open (เปิด)
- TH: ตรวจสอบรายละเอียด PR และเริ่มการรีวิว  
- EN: Review PR details and begin evaluation  

## ✅ Checks passed (ผ่านการตรวจสอบ)
- TH: ดำเนินการรีวิวโค้ดและ merge ได้เมื่ออนุมัติ  
- EN: Proceed with code review and merge once approved  

## ❌ Checks failed (ตรวจสอบล้มเหลว)
- TH: ตรวจสอบ log ของ pipeline, แจ้งผู้เขียนแก้ไข, ห้าม merge จนกว่าจะแก้ไขสำเร็จ  
- EN: Review pipeline logs, request author fixes, do not merge until resolved  

## 🟡 Pending review (รอการตรวจสอบ)
- TH: Reviewer ต้องให้ feedback หรือ approve/reject  
- EN: Reviewer must provide feedback or approve/reject  

## ⚪ Closed (ปิด)
- TH: ไม่ต้องดำเนินการต่อ, ใช้เพื่ออ้างอิงในอนาคต  
- EN: No further action, keep for reference  

## 🟣 Merged (รวมแล้ว)
- TH: ตรวจสอบว่า merge ตรงตาม governance rules และ log ถูกบันทึก  
- EN: Verify merge compliance with governance rules and ensure logs are recorded
