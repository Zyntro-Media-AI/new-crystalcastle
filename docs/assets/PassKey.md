📝**Passkeys and Security Keys (Google Account)**

**Summary**

Passkeys and security keys provide a secure, passwordless (or enhanced 2FA) method for signing into a Google account. 

### Key Features:
- **Authentication methods**: Use biometric authentication (fingerprint or face), device screen lock, or a physical security key.
- **Usage**:
  - Primary sign-in method (passwordless).
  - Can also serve as a second verification step when signing in with a password.
- **Security recommendations**:
  - Keep device screen lock information private.
  - Store physical security keys securely so that only you can access them.

### Creation Options
Passkeys can be created directly on a supported device or using a physical security key.

**Reference**: Google Account settings (myaccount.google.com) – Passkeys & Security Keys section.

This provides a convenient and phishing-resistant alternative to traditional passwords while maintaining strong security.
