Here’s the quick take from Brave’s help article:

What it is
• Widevine is the DRM plugin Brave uses to play protected video. Netflix, Hulu, HBO, Prime Video, etc. won’t stream without it   
It’s off by default
• After you install Brave on Android, Widevine is disabled. You turn it on in: Settings → Media → Widevine DRM (just toggle it)   
You’ll also get a prompt
• If you visit a DRM site first, Brave pops up a notice. Tap Allow (Relaunch Now) to enable and restart, or Block to keep it off — you can still change it later in Settings   
Requirements for HD (Widevine L1)
• Android N (7.0) or newer • Device with Trusted Execution Environment (TEE) hardware support • Stock/OEM OS — not rooted or heavily modified   
Brave links to their GitHub for deeper technical docs if you need to check your CDM version.
