# 🌟 SnapzReview Knowledge Base

ยินดีต้อนรับสู่คลังข้อมูลกลางสำหรับจัดการงาน Affiliate และระบบ Automation

---

## 🎯 วัตถุประสงค์
โปรเจกต์นี้สร้างขึ้นเพื่อรวบรวมองค์ความรู้ในการทำ **Content Creation** และการพัฒนา **Tools** สำหรับช่วยงาน Shopee Affiliate ให้มีประสิทธิภาพสูงสุด

## 📂 หมวดหมู่ความรู้

### 🛍️ [Affiliate & Content](./affiliate/strategy.md)
- กลยุทธ์การทำคอนเทนต์ใน Shopee Video, X และ Threads
- เทคนิคการเลือกสินค้าแฟชั่นชาย/หญิง
- แคปชั่นสำเร็จรูปสำหรับโพสต์ขายของ

### 💻 [Dev & Automation](./dev/automation.md)
- วิธีการตั้งค่าและรัน Python Scripts
- การจัดการ GitHub Actions และการส่งแจ้งเตือนผ่าน Line Notify
- คู่มือการใช้งาน API ต่างๆ

---

## 🚀 เริ่มต้นใช้งานรวดเร็ว (Quick Start)
1. อ่านสไตล์การทำงานในไฟล์ `.ai-context.md`
2. ตรวจสอบ [Setup Guide](./dev/setup.md) สำหรับการตั้งค่าระบบครั้งแรก
3. หากต้องการอัปเดตเอกสาร ให้เพิ่มไฟล์ `.md` ลงในโฟลเดอร์ `docs/knowledge/`

---
> **Note:** เอกสารนี้อัปเดตอัตโนมัติผ่าน GitHub Actions ทุกครั้งที่มีการ Push ข้อมูลใหม่เข้าสู่ Branch `main`
