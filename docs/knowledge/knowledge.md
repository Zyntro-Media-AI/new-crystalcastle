## 1. What Data This App Collects

| Category | Details |
|---|---|
| 📍 Location | GPS/precise location to show local weather; possibly background location even when app is closed |
| 📱 Device & Install | Device model, OS version, app install info, Advertising ID, device identifiers |
| 📊 Usage Behavior | App open frequency, features used, time spent in app |
| 📢 Advertising | Usage + location data shared with Facebook/Meta servers to build an ad profile |

### Key Finding
The app includes a **Facebook Analytics SDK** that:
- Collects user installation data
- Sends it to Facebook/Meta servers when the app is used
- Uses data to **support advertising**
- May link your activity to your Facebook/Instagram account via **Off-Meta Activity**

## 2. How to Manage Off-Meta Activity

### 📘 On Facebook
1. Open **Facebook** → Menu (☰) → **Settings & Privacy** → **Settings**
2. Go to **Your Facebook Information** → **Off-Facebook Activity**
3. Find **topfeed** or **สภาพอากาศและข่าวสาร** in the list
4. Tap **Disconnect Future Activity from Account**

### 📷 On Instagram
1. Open **Instagram** → Profile → ☰ → **Settings and Activity**
2. Go to **Your Account** → **Your Activity Off Meta**
3. Find the weather app in the list
4. Tap **Disconnect Future Activity**

### 🔄 Clear Past Data
- In the **Off-Meta Activity** screen, tap **Clear History**
- This unlinks previously collected data from your account
- ⚠️ Does **not** permanently delete data from Meta's servers

### 💡 Stop All Third-Party App Tracking
- Go to **Manage Future Activity** → Turn off **Future Off-Meta Activity**
- This blocks ALL third-party apps from linking data to your account
- The weather app will still work normally

## 3. Protect Your Privacy — Recommended Actions

| Action | How |
|---|---|
| Revoke location permission | Phone Settings → Apps → สภาพอากาศ → Permissions → Location → Deny or "Only while using" |

| Disable future tracking | Off-Meta Activity → Manage Future Activity → Turn off |

## 5. General Tips for App Privacy Audits

1. **Check app permissions** regularly (Location, Camera, Contacts, Microphone)
2. **Review Off-Meta Activity** on Facebook/Instagram monthly
3. **Read the "About" screen** in apps — it often discloses SDK usage
4. **Free apps with no ads** should raise questions about how they fund themselves — the answer is often your data
5. **Weather apps are high-risk** — they have a history of excessive data collection and selling location data to advertisers

## 7. Dropbox Desktop App — Installation Guide

> **Source:** https://help.dropbox.com/installs/download-dropbox  
> **Last Updated by Dropbox:** March 06, 2025

### Basic Install (Mac, Windows, Linux)
1. Go to **[dropbox.com/downloading](https://www.dropbox.com/downloading)**
2. Follow prompts to **log in or create an account**
3. Follow prompts to **sync your files**

After install, access Dropbox from:
- **Windows** → taskbar icon or File Explorer
- **Mac** → menu bar icon or Finder

### Install with Admin Privileges
The desktop app requires admin privileges for full functionality (sync icons, right-click menus).

1. Uninstall existing Dropbox first (if already installed)
2. Download installer from **dropbox.com/downloading**
3. Open installer — **do NOT right-click "Run as Administrator"**
4. Follow the wizard; enter admin credentials if prompted
5. Complete install and log in

> ⚠️ If on a work computer without admin rights, contact your IT department.

### Key Notes

| Topic | Detail |
|---|---|
| Windows 10 S Mode | Install **Dropbox Lite** instead |
| Network Drives | Dropbox does **not** support installation on network drives |
| Mac Keychain | Click **Always Allow** when prompted — required for Dropbox to start |
| System Extension | Auto-installed; allows opening online-only files from Finder/File Explorer |
| macOS 11+ | System extension does not apply; open online-only files directly in Finder |

### Reinstall Dropbox System Extension (if missing)
1. Log in to **dropbox.com**
2. Click your avatar → **Settings** → **General** tab
3. Toggle **Dropbox system extension** to **On**
4. Uninstall then reinstall the desktop app
