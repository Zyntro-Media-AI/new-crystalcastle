# 🎥 Camera Motion Library

## 🛸 Advanced Movements
- **Slow Push-In:** กล้องค่อยๆ เคลื่อนเข้าหาจุดเด่นของสินค้า เพื่อสร้างความรู้สึกพรีเมียมและน่าค้นหา.
- **360-Degree Rotation:** การหมุนรอบวัตถุแบบคงที่ เพื่อโชว์รายละเอียดสินค้าให้ครบทุกด้าน (เหมาะสำหรับ Catalog). [2]
- **Dynamic Handheld:** การจำลองกล้องถือด้วยมือที่มีการสั่นไหวเล็กน้อย เพื่อให้วิดีโอดูมีความเป็นมนุษย์และเข้ากับ TikTok/Reels. [2]

## 🛠️ Engine-Specific Tags
- **Kling/Runway:** ใช้คำสั่ง "Camera Pan" หรือ "Tilt" พร้อมระบุองศาที่ชัดเจนเพื่อความแม่นยำสูง. [6]
- **Sora:** เน้นการบรรยายการเคลื่อนที่แบบซับซ้อน (เช่น "Circling the subject while moving upward"). [6]
