🤖 Knowledge Document – Free AI Models Integration

🇹🇭 ภาษาไทย

โมเดล AI ฟรีที่สามารถใช้ได้
1. Meta LLaMA 3.1 (8B / 70B)  
   - โมเดลภาษาล่าสุดจาก Meta  
   - รองรับ multilingual, latency ต่ำ  
   - เหมาะสำหรับการสร้าง prompt และ QA  

2. Mistral 7B Instruct  
   - โมเดลเบา, เร็ว  
   - เหมาะสำหรับ fallback เมื่อ Groq ไม่พร้อมใช้งาน  

3. Microsoft Phi-3 Mini (4K / 128K)  
   - โมเดลขนาดเล็ก, latency ต่ำ  
   - รองรับ bilingual ได้ดี  

4. Google Gemma (2B / 7B)  
   - โมเดล lightweight  
   - เหมาะสำหรับ mobile/edge  

5. Whisper (OpenAI)  
   - ฟรีสำหรับ speech-to-text  
   - รองรับหลายภาษา  

6. LLaVA / MiniGPT-4  
   - ใช้สำหรับ multimodal (ภาพ + ข้อความ)  
   - เหมาะสำหรับ image captioning

🇬🇧 English

Free AI Models Available
1. Meta LLaMA 3.1 (8B / 70B)  
   - Latest language model from Meta  
   - Multilingual, low latency  
   - Ideal for prompt generation and QA  

2. Mistral 7B Instruct  
   - Lightweight, fast  
   - Suitable as fallback when Groq is unavailable  

3. Microsoft Phi-3 Mini (4K / 128K)  
   - Small model, low latency  
   - Strong bilingual support  

4. Google Gemma (2B / 7B)  
   - Lightweight model  
   - Optimized for mobile/edge  

5. Whisper (OpenAI)  
   - Free speech-to-text model  
   - Supports multiple languages  

6. LLaVA / MiniGPT-4  
   - Multimodal (image + text)  
   - Useful for image captioning
