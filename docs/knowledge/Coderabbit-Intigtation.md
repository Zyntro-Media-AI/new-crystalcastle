Overview
CodeRabbit is the AI code review platform for modern teams.

Works natively with Vercel projects. Context‑aware, customizable, and improving with every interaction. Catch bugs, harden security, and ship faster with one‑click fixes on every PR. Trusted by Open source projects like Bun, NuxtJS. Cut code review time and bugs in half.
