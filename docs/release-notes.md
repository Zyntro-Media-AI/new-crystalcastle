## 🔖 Version 1.2.0 — Governance Update
**English:**  
This release introduces improvements to governance enforcement, CI/CD workflows, and reviewer training artifacts.  

**ไทย:**

### 🚀 Features / ฟีเจอร์ใหม่
- **English:** Added bilingual PR template enforcing reviewer checklist

### 🛠 Fixes / การแก้ไข

- **English:** Deprecated files moved to `archive/` folder
