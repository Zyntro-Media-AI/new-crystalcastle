# 📜 CrystalCastle Changelog (Extended, Bilingual)

**English:**  
This extended changelog provides versioning rules, reviewer enforcement notes, and transparency logs for all releases.  

**ไทย:**  
Changelog แบบขยายนี้ให้กฎการจัดการเวอร์ชัน, หมายเหตุ reviewer และ transparency logs สำหรับทุก release

## 🔢 Versioning Rules / กฎการจัดการเวอร์ชัน
**English:**  
- **MAJOR (X.0.0):** Breaking changes, governance updates, or repo restructuring.  
- **MINOR (X.Y.0):** New features, workflow improvements, or documentation artifacts.  
- **PATCH (X.Y.Z):** Bug fixes, security patches, or minor workflow adjustments.  

**ไทย:**  
- **MAJOR (X.0.0):** การเปลี่ยนแปลงใหญ่, การอัปเดต governance หรือการปรับโครงสร้าง repo  
- **MINOR (X.Y.0):** ฟีเจอร์ใหม่, การปรับปรุง workflow หรือเอกสารใหม่  
- **PATCH (X.Y.Z):** การแก้บั๊ก, การแก้ไข security หรือการปรับ workflow เล็กน้อย

## 📝 Reviewer Enforcement Notes / หมายเหตุ Reviewer
**English:**  
- ❌ Reject PRs missing changelog updates.  
- ❌ Reject PRs with incorrect versioning (e.g., patch used for breaking change).  
- ✅ Approve PRs with clear bilingual changelog entries.  
- ✅ Ensure transparency log updated for governance-related changes.  

**ไทย:**  
- ❌ Reject PR ที่ไม่มีการอัปเดต changelog  
- ❌ Reject PR ที่ใช้ versioning ไม่ถูกต้อง (เช่น ใช้ patch สำหรับ breaking change)  
- ✅ Approve PR ที่มี changelog bilingual ชัดเจน  
- ✅ ตรวจสอบว่า transparency log ถูกอัปเดตสำหรับการเปลี่ยนแปลง governance
