# Passkey Authentication

## 🇹🇭 การยืนยันตัวตนด้วย Passkey
- ใช้ Passkey แทนรหัสผ่านเพื่อความปลอดภัยสูงขึ้น
- รองรับการเข้าสู่ระบบแบบ passwordless
- ลดความเสี่ยงจากการโจมตีด้วย phishing

## 🇬🇧 Passkey Authentication
- Use passkeys instead of passwords for stronger security
- Support passwordless login
- Reduce risk of phishing attacks
