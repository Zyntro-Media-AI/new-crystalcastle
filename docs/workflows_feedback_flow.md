# Feedback Flow (English / ภาษาไทย)

## 📌 Purpose / วัตถุประสงค์
- Provide a structured way for team members to submit and review feedback  
- สร้างมาตรฐานการส่งและตรวจสอบ feedback ให้ทีมใช้ร่วมกัน  

---

## 📝 Feedback Template / เทมเพลต Feedback

### English
- **Feedback Type**: [Workflow / Code / UI / Governance]  
- **Summary**:  
  - Point 1  
  - Point 2  
- **Suggested Action**:  
  - Recommendation  

### ภาษาไทย
- **ประเภท Feedback**: [Workflow / Code / UI / Governance]  
- **สรุป**:  
  - ประเด็นที่ 1  
  - ประเด็นที่ 2  
- **ข้อเสนอแนะ**:  
  - แนวทางแก้ไข
