# Knowledge Management – Repository Level

## 🇹🇭 การจัดการความรู้ในระดับ Repository
- จัดโครงสร้างโฟลเดอร์ docs/ สำหรับ reviewer artifacts
- ใช้ commit message bilingual เพื่อ repo hygiene
- บังคับใช้ reviewer checklist และ reviewer notes ทุกครั้งที่ merge

## 🇬🇧 Knowledge Management at Repository Level
- Organize docs/ folder for reviewer artifacts
- Enforce bilingual commit messages for repo hygiene
- Apply reviewer checklist and reviewer notes before every merge
