# Security Policy

## Supported Versions

ตอนนี้โปรเจกต์ยังเป็น Private/Experimental ใช้ส่วนตัว

| Version | Supported |
| --- | --- |
| main | ✅ |

## Reporting a Vulnerability

ถ้าเจอช่องโหว่ความปลอดภัย เช่น XSS, ข้อมูลหลุด, API Key รั่ว

**ติดต่อ:** เปิด [GitHub Security Advisory](https://github.com/1napz/crystalcastle/security/advisories/new)

เราจะตอบกลับภายใน 48 ชั่วโมง และจะแจ้งความคืบหน้าจนกว่าจะแก้เสร็จ

ขอบคุณที่ช่วยกันดูแลให้ปลอดภัย

## Out of Scope
- Rate limiting
- Issues ที่ต้องการ physical access
- Social engineering
