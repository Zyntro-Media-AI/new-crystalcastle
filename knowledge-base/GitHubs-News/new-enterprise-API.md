GitHub has released a new **Enterprise Installation API** in public preview, addressing a gap that the GitHub Actions community had flagged. The new API lets a GitHub App check whether it's installed on an enterprise and retrieve its installation ID — bringing enterprise support in line with equivalent APIs that already exist for organizations, repositories, and users. The main practical benefit is faster acquisition of installation tokens for an enterprise, without having to paginate through all installations to find the right one. Feedback can be left in the linked Community discussion. [github](https://github.blog/changelog/2026-05-13-new-enterprise-installation-api-now-in-public-preview/)
