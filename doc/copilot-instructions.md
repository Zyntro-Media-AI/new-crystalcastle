## Copilot Instructions for Crystal Castle

- Use Vanilla JavaScript (ES Modules)
- Prefer `innerHTML` over `createElement` for DOM manipulation
- Do NOT use jQuery
- Use Tailwind CSS for styling
- API Routes are in `/api/*.js`
- Supabase client is initialized in `supabase-client.js`
- Always handle errors with try/catch
- Always add AbortController to fetch requests
