# 📚 Crystal Castle Documentation

ยินดีต้อนรับสู่ศูนย์รวมเอกสารของโปรเจกต์ Crystal Castle 🏰  
AI Video Studio ที่ใช้ Supabase เป็น Backend, Vercel สำหรับ Deploy และ Groq สำหรับสร้าง Prompt

## 🛠️ ภาพรวมโปรเจกต์

- **Frontend**: Static HTML + Tailwind CSS + Vanilla JS
- **Backend**: Next.js API Routes (Serverless Functions บน Vercel)
- **Database & Storage**: Supabase
- **AI**: Groq, FAL Kling, Magic Hour, OpenRouter (ทางเลือก)
- **Deploy**: Vercel (เชื่อมกับ GitHub)
