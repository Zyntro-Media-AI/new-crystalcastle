# 🎬 Crystal Castle Lite v1.3

**เครื่องเจียระไน Lite** — อัพรูปสินค้า → AI แต่ง Prompt → เจนวิดีโอ → สร้างแคปชั่นอัตโนมัติ

Live: https://crystalcastle-pi.vercel.app/product.html

---

## ✨ มีอะไรใหม่ใน v1.3

- ✅ **UI มือถือแก้แล้ว** - textarea ไม่บีบ ปุ่มอยู่ใต้บนจอเล็ก
- ✅ **Gen Prompt ด้วย Groq** (Llama 3.3 70B) - เร็ว 2-3 วิ ฟรี
- ✅ **Generate Post อัตโนมัติ** - ได้แคปชั่นไทย + hashtag พร้อมโพสต์
- ✅ **2 Engine**: FAL Kling (เร็ว 1.8 บาท) / Magic Hour (ฟรี)
- ✅ **Auto filename** ตามวันที่จริง `yyyymmdd-Category-Brand`

## 🚀 วิธีใช้งาน

1. **เลือกรูปสินค้า** - JPG/PNG
2. **ใส่ Category + Brand** - ระบบตั้งชื่อไฟล์อัตโนมัติ
3. **กด ✨ Gen Prompt** - AI แต่ง prompt วิดีโอแฟชั่นให้
4. **กด 📝 Generate Post** - ได้แคปชั่นภาษาไทยพร้อม hashtag
5. **เลือก Engine** แล้วรอวิดีโอ
