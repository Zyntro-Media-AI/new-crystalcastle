## 🎯 What is Crystal Castle?

Crystal Castle คือระบบ **AI Video Generator สำหรับ Social Commerce**

เปลี่ยน:
- 📸 Product Image  
→ 🎬 AI Video  
→ 📝 Auto Caption  
→ 🚀 พร้อมโพสต์ขายทันที

---

## ⚡ Key Features

- 🎬 AI Video Generation (Image → Video)
- 🧠 Multi-Engine AI Router
- ✨ Crystal Light & Glass Effects
- 📝 Auto Caption (Marketing-ready)
- ⚡ Fast Render Pipeline

🛠️ Tech Stack

Frontend: HTML / CSS / JS

AI Layer: Custom Engine Router

Backend: Node.js

Infra: Vercel + Supabase
