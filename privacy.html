Privacy Policy — Crystal Castle

อัปเดตล่าสุด: 23 เมษายน 2026

1. ข้อมูลที่เราเก็บรวบรวม

Crystal Castle เก็บรวบรวมข้อมูลดังต่อไปนี้:

  • ข้อมูลบัญชีผู้ใช้: อีเมล (สำหรับเข้าสู่ระบบผ่าน Supabase Authentication)
  • รูปสินค้า: รูปภาพที่คุณอัปโหลดเพื่อใช้สร้างวิดีโอ (จัดเก็บใน Supabase Storage)
  • Prompt และวิดีโอที่สร้าง: ข้อความคำสั่งและวิดีโอที่ระบบ AI สร้างให้ (เก็บไว้ในประวัติการใช้งาน)
  • ข้อมูลการใช้งาน: บันทึก Logs (engine, status, timestamp) เพื่อการพัฒนาและปรับปรุงบริการ

2. การใช้ข้อมูล

เราใช้ข้อมูลของคุณเพื่อ:

  • ยืนยันตัวตนและจัดการบัญชีผู้ใช้
  • สร้างวิดีโอจากรูปสินค้าตามที่คุณร้องขอ
  • ปรับปรุงคุณภาพของ AI (Groq, FAL Kling, Magic Hour)
  • วิเคราะห์ประสิทธิภาพและการใช้งานระบบ

3. การจัดเก็บข้อมูล

ข้อมูลทั้งหมดจัดเก็บผ่าน Supabase (ผู้ให้บริการคลาวด์) บนเซิร์ฟเวอร์ในสหรัฐอเมริกา โดย:

  • รูปสินค้าและวิดีโอ: เก็บไว้ใน Storage (สูงสุด 1GB สำหรับบัญชีฟรี)
  • Logs การใช้งาน: เก็บไว้ในฐานข้อมูล PostgreSQL (เก็บไว้สูงสุด 90 วัน)
  • ข้อมูลบัญชี: จัดการโดย Supabase Auth (คุณสามารถลบบัญชีได้ตลอดเวลา)

4. การแบ่งปันข้อมูล

เรา ไม่ขาย ไม่แลกเปลี่ยน และไม่เปิดเผย ข้อมูลส่วนตัวของคุณให้บุคคลที่สาม เว้นแต่:

  • เป็นการร้องขอโดยชอบด้วยกฎหมาย (ศาล, ตำรวจ)
  • จำเป็นเพื่อดำเนินการตามคำขอของคุณ (เช่น ส่งรูปไปยัง API ของ FAL Kling เพื่อสร้างวิดีโอ)
  • เพื่อปกป้องสิทธิ์ ความปลอดภัย หรือทรัพย์สินของเรา

5. การรักษาความปลอดภัย

เรามีมาตรการรักษาความปลอดภัยที่เหมาะสม เช่น:

  • ใช้ HTTPS สำหรับการสื่อสารทั้งหมด
  • ใช้ Row Level Security (RLS) ใน Supabase เพื่อควบคุมการเข้าถึงข้อมูล
  • ไม่เก็บ API Keys หรือ Secrets ไว้ในโค้ดฝั่งไคลเอ็นต์

6. สิทธิ์ของคุณ

คุณมีสิทธิ์:

  • เข้าถึง แก้ไข หรือลบข้อมูลส่วนตัวของคุณ
  • ขอรับสำเนาข้อมูลที่เราจัดเก็บ
  • ยกเลิกการใช้บริการและลบบัญชีได้ตลอดเวลา (ติดต่อผ่านอีเมล)

7. การเปลี่ยนแปลงนโยบาย

เราอาจอัปเดตนโยบายนี้เป็นครั้งคราว หากมีการเปลี่ยนแปลงสำคัญ เราจะแจ้งให้ทราบผ่านทางอีเมลหรือประกาศบนเว็บไซต์
