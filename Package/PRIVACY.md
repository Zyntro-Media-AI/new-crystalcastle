# Privacy Policy

Crystal Castle respects your privacy:

- No personal data is stored in the repository.
- API keys are required for AI services (Groq, Gemini, FAL, Magic Hour) and must be kept private.
- Supabase is used for storage; Row Level Security (RLS) is enabled.
- Logs are anonymized and used only for debugging and performance monitoring.
- We do not sell or share user data with third parties.
